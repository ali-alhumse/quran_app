# Quran App

تطبيق قرآن بسيط مبني باستخدام React Native و Expo. يجلب التطبيق قائمة السور والآيات من واجهة [Al Quran Cloud API](https://alquran.cloud/api)، ويعرضها بواجهة عربية داعمة للقراءة من اليمين إلى اليسار.

## المميزات

- جلب قائمة السور من API خارجي.
- البحث باسم السورة، الاسم الإنجليزي، أو رقم السورة.
- عرض آيات السورة المختارة بالنص العربي.
- حالات تحميل، رسائل خطأ، وإعادة محاولة.
- واجهة عربية نظيفة مناسبة للهاتف.

## التشغيل

ثبت الاعتمادات ثم شغل Expo:

```bash
npm install
npm start
```

بعد ذلك افتح التطبيق عبر Expo Go أو محاكي Android/iOS.
